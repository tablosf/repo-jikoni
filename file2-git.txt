--- 9.2 Setting DNS di Client Windows ---
1. Buka Control Panel > Network and Sharing Center
2. Klik adapter yang dipakai (Ethernet/WiFi)
3. Klik Properties
4. Pilih Internet Protocol Version 4 (TCP/IPv4) > Properties
5. Pilih "Use the following DNS server addresses" dan juga gateway
6. Preferred DNS server: 192.168.1.11
7. Klik OK
