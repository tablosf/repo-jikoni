7.2 Inisialisasi & Push ke GitHub
cd /repo-ujikombengkel

git remote add origin https://github.com/username/nama-repo.git
git branch -M main

git clone https://github.com/ershnir/repo-ujikombengkel.git
cd repo-ujikombengkel
7.4 Workflow Git Sehari-hari
•	git status — cek perubahan
•	git add . — stage semua perubahan
•	git commit -m 'pesan' — commit perubahan
•	git push origin main — upload ke GitHub
•	git pull origin main — update dari GitHub
